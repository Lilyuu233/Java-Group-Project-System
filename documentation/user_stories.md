## User Stories


### Operator

As Tammie Jycross, the Operator, I want to select the dataset to compress, so that I can select the uncompressed data from the IP App Store.


### Control Systems Engineer


As Nuce Bricholson, the Control Systems Engineer, I want to configure compression parameters in a way that will reduce storage space used, so that we can store more data for cheaper.

As Nuce Bricholson, the Control Systems Engineer, I want to view a data metrics for different compression parameter configurations, so that I can choose the configuration which reduces storage the most.

As a control systems engineer I want to be able to mark data for deletion in the historian once I have compressed the data and moved it somewhere else.

as a control systems engineer I want to be able to delete data from the historian once I have verified with the operator and systems engineer that the compressed data is good enough for their work.

### Systems Engineer


As Plosh Jumbly, the Systems Engineer, I want to compress data in a way that will retain as much accuracy as possible, so that our system can be calibrated with the most accuracy possible.

As Plosh Jumbly, the Systems Engineer, I want to view a graphical comparison of the effects of different compression parameter configurations, so that I can choose the configuration which retains the most accuracy.

### Other

As a user I don't want to be compressing already compressed data.

As a user, I don't want to be compressing the same raw data over and over again.
