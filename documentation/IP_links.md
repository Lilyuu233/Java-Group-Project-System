#### Notes
- There is a mix of resources in C#, JavaScript, TypeScript, R and using Jupyter Notebook
- The Industrial App Store provides clients with: a shop front to market the app, a user authentication API, a monetisation API (so that customers can be charged) and a data API (to allow secure connection to authroised customer data sources)
- The app store can market websites, desktop programs, spreadsheets, services, models etc. and can be built on any platform that supports secure connection with the app store API
- Access tokens are used with API calls to authenticate and authorise the calling user and app combination
- Data requests are executed via the Data Core API
- Specific detail and available functions when querying the dataset is available on the data request link above
